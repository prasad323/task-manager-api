# Task Manager REST API

Simple Task Manager backend using Node.js, Express, MongoDB and JWT authentication.

## Requirements
- Node 22.14.0
- MongoDB running locally or connection string
- npm
-nodemon

## Setup
1. Clone repository
2. Copy  `.env` and fill values:
   - `MONGO_URI`, `JWT_SECRET`, `PORT` 
3. Install:
   ```bash
   npm install
